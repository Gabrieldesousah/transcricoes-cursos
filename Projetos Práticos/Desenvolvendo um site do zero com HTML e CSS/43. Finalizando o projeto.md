# Finalizando o projeto

Em primeiro lugar acertaremos o menu lateral em relação ao conteúdo do chat, pois o conteúdo está maior do que o menu e isso faz com que o menu quebre o layout em um determinado momento.

Primeiro adicione o seguinte css no arquivo **projeto/css/styles.css**:

```css
html, body, .box-100-height {
    height: 100%;
}

#sidenav {
    background-color: #333333;
    color: #888;
    padding-right: 0;
    overflow-y: auto;
}

#content {
    overflow-y: auto;
}
```

A princípio esta alteração não modificará nada em nosso projeto, mas adicionando a classe **box-100-height** aos blocos principais nós acertaremos este problema. Veja as inserções no HTML:

```html
<!-- Adicionando classe no bloco sidenav -->
<div class="container-fluid box-100-height">
    <div class="row box-100-height">
        <section id="sidenav" class="col-md-2 box-100-height">
			<!-- código -->
        </section>
    </div>
</div>

<!-- Adicionando classe no bloco content -->
<section id="content" class="col-md-10 box-100-height">
	<!-- código -->
</section>
```

Após fazer estas alterações e inclusões, você já pode atualizar seu navegador e não haverá mais quebra de layout, porque tanto o content quando o menu estão sendo limitados ao tamanho da tela. Tudo que passar deste tamanho será ocultado e será criada uma barra de rolagem para a exibição do conteúdo excedente.

O próximo passo será melhorar o menu superior do content:
