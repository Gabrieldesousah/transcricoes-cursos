```css
@import url('../styles.css');

#header.page-clientes {
    background-image: url("../../images/background-clientes.jpg");
}
```

Para este momento de nosso projeto, estas alterações são suficientes para que o conteúdo seja exibido ao acessar o link de clientes.

Não se esqueça de criar as imagens, no caminho correto, ou copiar as imagens do código fonte, para que as imagens sejam mostradas corretamente.
